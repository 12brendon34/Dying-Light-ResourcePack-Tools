This repository provides tools for extracting and working with game resource files used in `Dying Light: The Beast`.

---
## Dying Light `.rpack` Extractor
The extractor can unpack and dump file formats contained in `.rpack` archives:

- **.DDS** — Texture files  
- **.MSH** — Mesh files  
- **.ANM2** — Animation files  
- **Other formats** — Dumped as raw binary data
